## Table: Activities

| Column Name | Type    |
|-------------|---------|
| sell_date   | date    |
| product     | varchar |

- There is no primary key for this table; it may contain duplicate rows.
- Each row contains the product name and the date it was sold in a market.

---

## Problem Statement

Write a solution to find **for each date**:
- the **number of different products sold**
- the **names of the products sold on that date**

The product names for each date should be:
- **distinct**
- **sorted lexicographically**
- **separated by commas**

Return the result table ordered by **sell_date**.

The result format is shown in the following example.

---

## Example 1

### Input

**Activities table:**

| sell_date  | product     |
|------------|-------------|
| 2020-05-30 | Headphone   |
| 2020-06-01 | Pencil      |
| 2020-06-02 | Mask        |
| 2020-05-30 | Basketball  |
| 2020-06-01 | Bible       |
| 2020-06-02 | Mask        |
| 2020-05-30 | T-Shirt     |

---

### Output

| sell_date  | num_sold | products                      |
|------------|----------|-------------------------------|
| 2020-05-30 | 3        | Basketball,Headphone,T-Shirt  |
| 2020-06-01 | 2        | Bible,Pencil                  |
| 2020-06-02 | 1        | Mask                          |

---

### Explanation

- For `2020-05-30`, sold products were `(Headphone, Basketball, T-Shirt)`.
- For `2020-06-01`, sold products were `(Pencil, Bible)`.
- For `2020-06-02`, the only sold product was `(Mask)`.

Products are sorted lexicographically and joined using commas.

---

## Answer

```sql
select sell_date,
       count(distinct product) as num_sold,
       group_concat(distinct product order by product separator ',') as products
from Activities
group by sell_date
order by sell_date;
