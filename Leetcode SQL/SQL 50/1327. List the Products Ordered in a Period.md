## Problem

Find products that:

- Were ordered in **February 2020**
- Have **total units ≥ 100**
- Return: product_name and total units

---

## Example Understanding

👉 For each product:

- Filter only Feb 2020 orders  
- Sum units  
- Keep only ≥ 100  

---

## Logic

1. Join Products and Orders  
2. Filter February 2020  
3. Group by product  
4. Sum units  
5. Apply HAVING ≥ 100  

---

## Answer

```sql
SELECT P.product_name, SUM(O.unit) AS unit
FROM Products P
JOIN Orders O
ON P.product_id = O.product_id
WHERE O.order_date BETWEEN '2020-02-01' AND '2020-02-29'
GROUP BY P.product_id, P.product_name
HAVING SUM(O.unit) >= 100;

-- OR

SELECT P.Product_name, SUM(O.unit) as unit
FROM Products P
JOIN Orders O
ON P.product_id = O.product_id
WHERE MONTH(order_date) = 2 AND YEAR(order_date) = 2020
GROUP BY P.product_id, P.product_name
HAVING SUM(O.unit) >= 100;
